Overview

The Tenant Guide Setup module is a powerful documentation builder that allows administrators to create and maintain instructional resources for platform tenants. This ensures that every user has access to a structured knowledge base directly within their dashboard.

Admins can categorize guides, reorder them for better flow, and build content-rich pages using sections and multimedia placeholders.

Managing the Guide Menu

The structural hierarchy of your documentation is managed through an interactive sidebar menu:

  • Nested Categories: Organize guides into logical groups. Categories can be expanded or collapsed to keep the workspace clean.
  • Dynamic Reordering: Every guide and category features a drag-handle icon. Simply drag an item to change its position in the tenant-facing navigation.
  • Instant Search: Use the search bar at the top of the menu to find specific guides by title, even in large documentation sets.
  • Global Creation: Click the green (+) icon at the top right of the menu to start building a new guide from scratch.

Building Guide Content

The editor panel provides a comprehensive set of tools to craft your documentation with precision:

Guide Metadata

Define the primary Title and Subtitle that anchor the guide's identity for tenants.

Category Logic

Sort guides into groups or create a + new category instantly to maintain a clean structure.

Active Control

Toggle the Active Status switch to instantly publish or archive documentation.

Rich Editor

Use the Main Description block for formatted text, links, and high-impact messaging.

Content Segments (Guide Sections)

For complex topics, use the Guide Sections module to break content into logical titled blocks. This creates a paginated-style flow, ensuring tenants aren't overwhelmed by long scrolls of text.

Guide Sections

For complex topics, you can break down a single guide into multiple Sections. Each section acts as a sub-topic with its own title and content block, allowing for a structured reading experience.

Pro Tip: Content Flow

Use sections to separate technical steps from general overviews. This makes the documentation easier to scan for tenants.

Advanced Organization

WAPI provides a highly flexible drag-and-drop system to ensure your documentation follows a logical learning path:

Category Reordering

Adjust the sequence of entire documentation groups. Simply grab the handle icon next to a category name and drag it to your preferred position. This reorders the main navigation menu for all tenants.

Guide Sequencing

Within each category, you can reorder individual guides. This allows you to create a "Step 1, Step 2" flow, ensuring tenants encounter information in the correct order of importance.

Note: Reordering takes effect instantly across all tenant dashboards. No manual "Publish" is required for structural sequence changes. click on save changes will directly save the reordered list.

How to Access

1

Log in to the WAPI Admin Panel.

2

Navigate to Asset Management in the primary sidebar.

3

Click Tenant Guide Setup to launch the manager.
